This project addresses the problem of unstructured and untracked support requests in organizations. Support issues often arrive through emails, messages, or verbal communication, making it difficult to track ownership, status, and resolution history.

The Lightweight Support Ticket Management System provides a centralized and structured solution where every issue is logged as a ticket and tracked through its complete lifecycle.
